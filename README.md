# batscream

Browser game prototype: navigate a cave by sound. Scream to emit echolocation pulses, turn your head to steer, and collect glowing bugs.

## Run

Serve the folder locally and open `echolocation.html` in a modern browser over `http://localhost`.

Example:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000/echolocation.html`.
